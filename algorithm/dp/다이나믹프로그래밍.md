# 다이나믹 프로그래밍
여러개의 하위 문제를 먼저 풀고, 그 결과를 쌓아 올려, 주어진 문제를 해결하는 알고리즘 

## DP를 푸는 방법
- 테이블 정의하기
- 점화식 찾기
- 초기값 설정하기

### 떠올려야되는 생각들

#### 테이블 정의하기
> 최종적으로 무엇을 구해야하는가?  
> 인덱스로 어떻게 표현할 수 있는가?

#### 점화식 찾기
> 이전 결과로 현재를 표현할 수 있는가?  
> 이전 값들과 현재값들의 관계는 뭐지?

#### 초기값 설정하기
> 점화식으로 정할 수 없는 초기값들은 뭐지?


#### 풀이 팁
- 테이블을 설정할때, i를 최종 답이라 생각하고 접근해보자.  
ex) dp[i] = i번째까지의 최솟값,   
그리고 이 값들이 모여서 최종목적지인 X가 됐을때, dp채우기를 종료하는 느낌이다.  
- dp[i]에 도달하기 위해선 그 이전에 채워진 dp값에서 어떤 경우의 수를 거쳐서 도달하는지 고려해보면 점화식 작성하기가 수월하다.
