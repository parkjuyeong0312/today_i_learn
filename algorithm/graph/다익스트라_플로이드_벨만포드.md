
# 다익스트라 알고리즘
### 특징
다익스트라는 **한 지점에서 시작 하여 모든 지점까지의 최단거리**를 구하는 알고리즘이다.
greedy 하게 작동한다.
오직 양수 가중치에서만 작동한다는 특징이 있다.  
시간 복잡도는 EO(V)이다.  
우선순위 큐를 사용한다.  

### 구현방법
1. 각 거리가 되는 distance 테이블을 만들고, 모두 무한대로 초기화한다.
    - 그 이유는, 최초로 도착하는 distnace가 할당되어야 되기 때문이다.
2. 시작점을 priorityQueue에 넣는다.
3. queue에서 node를 꺼낸다. (index, distance)를 얻는다.
4. distance[index] < distance 라면, 즉, 최솟값을 갱신시키지 못했다면 continue한다.
5. 해당 노드와 이웃한 노드들을 나열하고, cost = distance[now]+neighbor.distance라고 했을때,  
cost < distance[index]라고 했을 때, 즉, 최단거리를 갱신했을때, pq에 추가로 넣는다.
