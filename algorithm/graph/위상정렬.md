## 위상정렬
'사이클이 없는' 방향그래프에서 간선으로 주어진 정점 간 **선후관계**를 위배하지 않도록 나열하는 정렬

### 구현방법
1. 맨 처음 모든 간선을 읽으며, indegree(노드로 들어오는 간선의 개수) 테이블을 만든다.
2. Indegree가 0인 정점들을 모두 저장 큐에 넣는다.
3. 큐에서 정점을 꺼내서 위상정렬 결과에 추가한다.
4. 정점으로부터 연결된 모든 정점의 indegree값을 1 감소시킨다. 이 때 indegree가 0이 되었다면, 그 정점을 큐에 추가한다. 
5. 큐가 빌 때 까지 3,4번 과정을 반복한다.

```java
import java.util.*;

public class TopologicalSort {
    public static void main(String[] args) {
        // [변수 선언부] 이미지의 1~3번째 줄
        int n = /* 노드의 총 개수 */; 
        
        // C++의 vector<int> adj[10] 에 해당 (인접 리스트)
        List<Integer>[] adj = new ArrayList[10];
        for(int i = 0; i < 10; i++) {
            adj[i] = new ArrayList<>();
        }
        
        // C++의 int deg[10] 에 해당 (진입차수 배열)
        int[] deg = new int[10]; 
        
        // (참고: 이곳에 그래프 간선 정보 및 진입차수를 채우는 로직이 있다고 가정합니다)

        // [알고리즘 구현부] 이미지의 5~21번째 줄
        Queue<Integer> q = new LinkedList<>();   // 5번 줄: 큐 선언
        List<Integer> result = new ArrayList<>(); // 6번 줄: 결과를 담을 리스트

        // 1. 초기화: 진입차수가 0인 노드를 모두 큐에 삽입 (7~8번 줄)
        for (int i = 1; i <= n; i++) {
            if (deg[i] == 0) {
                q.offer(i); 
            }
        }

        // 2. 위상 정렬 진행 (9~16번 줄)
        while (!q.isEmpty()) {
            int cur = q.poll(); // 큐의 맨 앞 원소를 빼옴 (C++의 front + pop)
            result.add(cur);    // 결과 리스트에 현재 노드 추가

            // 현재 노드와 연결된 인접 노드들을 확인
            for (int nxt : adj[cur]) {
                deg[nxt]--; // 인접 노드의 진입차수를 1 감소시킴
                
                // 진입차수가 0이 되었다면 큐에 삽입
                if (deg[nxt] == 0) {
                    q.offer(nxt);
                }
            }
        }

        // 3. 사이클 존재 여부 판별 및 결과 출력 (17~21번 줄)
        if (result.size() != n) {
            // 큐에 들어갔다 나온 노드의 수가 전체 노드 수와 다르면 사이클이 존재한다는 의미
            System.out.println("cycle exists");
        } else {
            // 사이클이 없다면 위상 정렬 결과 출력
            for (int x : result) {
                System.out.print(x + " ");
            }
        }
    }
}
```
