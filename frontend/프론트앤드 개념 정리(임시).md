# Node.js란?
- Node.js는 Chrome V8 JavaScript 엔진으로 빌드된 JavaScript 런타임입니다.
- 런타임은 특정 언어로 만든 프로그램을 실행할 수 있는 환경을 뜻함.


# html이란?
- HyperText Markup Languag
- <div>, <h1>, <p> 같은 태그를 사용해서 제목, 문단, 이미지, 버튼 등을 배치
- 웹 브라우저에게 여기는 제목이고, 본문이야 라고 알려주는 역할을 함.
- 텍스트가 아니고, "하이퍼 텍스트"인 이유는?
    - 여기서 하이퍼는 초월의 의미로, 연결을 가능하게 됨. 
    - 브라우저는 여기서 html을 읽어서, 랜더링을 해줌.

# javascript(언어)
- 웹 페이지를 동적으로 만들기 위해 설계된 프로그래밍 언어이다.
