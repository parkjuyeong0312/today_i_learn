# Node.js란?
- Node.js는 Chrome V8 JavaScript 엔진으로 빌드된 JavaScript 런타임입니다.
- 런타임은 특정 언어로 만든 프로그램을 실행할 수 있는 환경을 뜻함.


# html이란?
- HyperText Markup Languag
- <div>, <h1>, <p> 같은 태그를 사용해서 제목, 문단, 이미지, 버튼 등을 배치
- 웹 브라우저에게 여기는 제목이고, 본문이야 라고 알려주는 역할을 함.
- 브라우저는 여기서 html을 읽어서, 랜더링을 해줌.

# javascript(언어)
- 웹 페이지를 동적으로 만들기 위해 설계된 프로그래밍 언어이다.
- html의 언어에 종속(따까리)로써, 동적인 부분을 처리해준다.

# NodeJS란?(Runtime 환경)
크롬의 v8 엔진(다른 브라우저와 비교했을때 우수함, 매우 빠름)을 떼와서 브라우저가 아닌 환경에서 javascript을 구동할 수 있게 만든 런타임 환경이다.
즉, 원래같으면, javascript개발등을 한다고해도, 디버깅이나, 개발을 브라우저에서만 할 수 있었겠지만, 이것을 
따로 빼와서, npm run 과 같이 cli상에서 작업할 수 있게 된 것이다.


## npm이란?(패키지 매니저)
- Node packaged Manager (NPM)
- Node.js로 만들어진 모듈을 웹에서 받아서 설치해주고 관리해주는 프로그램
- 이 모듈을 사용하면, 만약 사용하고 있는 모듈이 업데이트가 되면, 체크해서 알려줌

npm install을 하게 되면, 
package.json이라는 파일의 dependecies에 해당하는 라이브러리들을 받아와서
node_modules에 저장하고, 이것을 가져다 쓸 수 있게 된다.

npm run serve -> build 하는 명령어.
