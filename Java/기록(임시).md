# Optional
Java 8 버전 이전에는 값이 없는 경우 보통 null을 반환했음.
하지만 null을 반환하게 되면 이를 참조하는 과정에서 NullPointerException(NPE)이 발생하게 됨.
이를 방지하고자 **Optional**이라는 반환을 목적으로 한 컨테이너 클래스가 등장함.

Optional에서는 isPresent(), isEmpty() 같은 메서드를 통해 값이 있는지 안전하게 확인할 수 있고, 
orElse()와 같이 만약 null이면 다른 값을 반환할 수 있는 기능을 제공함.
Optional의 가장 큰 특징은 Optional 객체 자체에 null을 할당하면 안 된다는 것임. 
따라서 값이 비어있을 때는 Optional.empty()를 반환하거나 위에서 언급한 메서드들을 사용해서 적절한 처리를 해줘야 함.
